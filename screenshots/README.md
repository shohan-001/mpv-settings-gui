# Screenshot Guidelines

To make your repository look highly professional and clear for users, please capture and save the following screenshots into this `screenshots/` directory using these exact filenames:

1. **`dashboard.png`**
   - **What to capture**: The main dashboard screen showing the "Video" or "Audio" settings tab, displaying the premium dark-themed layout, responsive cards, and search bar.
   
2. **`playlist_manager.png`**
   - **What to capture**: The "Playlist Manager" tab showing multiple media files added to the queue, showing the Up/Down reordering arrows and the "Launch Playlist in MPV" button.

3. **`controls_guide.png`**
   - **What to capture**: The "Controls Guide" tab showing the basic/advanced keyboard shortcuts list with styled keyboard badges (`<kbd>`).

4. **`key_bindings.png`**
   - **What to capture**: The "Key Bindings" editor tab showing the list of custom keys mapping to MPV commands, highlighting search/filter inputs.

5. **`suggestions_modal.png`**
   - **What to capture**: The "Add/Edit Hotkey" modal with the MPV Command Suggestions dropdown open, showing templates like "Volume control", "Seek playback", etc.

6. **`backups.png`**
   - **What to capture**: The "Backups History" panel dropdown showing the lists of timestamped configuration file backups.

Once you save these images in this folder, they will render automatically on your main repository page!
